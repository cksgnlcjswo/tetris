1. mainMenu 함수를 실행.

2. mainMenu의 선택에 따라 적절한 함수를 실행. (게임시작, 스코어 보드, 게임 종료)

    2.1 만약 게임 실행의 선택을 했을경우
        - gameStart()라는 함수를 통해 게임을 실행한다.(현재 우리가 main.cpp에서 작업하던 내용을 옮긴 함수라는 가정.)
        - 해당 함수 실행 후 3가지의 경우가 갈린다.

                2.1.1 게임이 그냥 진행될 경우. (특별한 케이스 없음. 그냥 그대로.)

                2.1.2 게임이 중간에 멈출경우. (pauseMenu() 함수를 통해 새로운 인터페이스를 띄운다.)
                    - pauseMenu()에는 2가지 선택지가 있다.
                        ;) 게임 종료.
                            - 게임 종료. (window.close() 를 통해 window를 끈다.)
                        ;;) 게임 계속하기.
                            - 다시 게임으로 돌아간다. (그냥 해당 함수 종료하면 되니까 return 해주면 될듯.)

                2.1.3 게임 오버했을 경우.
                    - gameOver() 함수를 통해 game over 시의 행동을 취함.
                    - score를 입력해야 하므로, 플레이어 명을 입력받고 score를 기록하는 창을 띄운 후 입력을 받는다a.
                    - 기록 후 mainMenu()로 돌아감.
                    - 현재 stackframe의 상태는 main() -> mainMenu() -> gameStart() -> gameOver() : 현재 위치.
                    - gameStart()에서 gameOver()후에 return 을 해서 mainMenu()로 돌아가게 하자.

    2.2 만약 스코어 보드를 선택했을 경우.
        - scoreBoard()라는 함수를 통해 스코어 보드를 보여준다.
        - 스코어 보드 상에서 mainMenu()로 복귀하기 위한 키를 알려준다.
        - mainMenu()로의 복귀는 return을 통해서 한다.

    2.3 만약 게임 종료를 선택시.
        - 게임을 종료한다.
